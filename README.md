# Sketch Gallery

Sketch Gallery is an Open Source public art gallery, built with [Flask](http://flask.pocoo.org/) (Backend)
and [EmberJS](http://emberjs.com/) with [Sketchpad](http://yiom.github.io/sketchpad/) (Frontend).
